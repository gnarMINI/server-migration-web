Kakao IDC 서버 이전 리스트 [CBT]

  • 1. 작업 등록: 이전 종류, 센터, 서버 수, 벤더, 요청자 등 필수 항목을 입력한 뒤 SAVE 버튼을 클릭하여 등록한다.
  • 2. 이전 유형 선택: 센터 간 이전 또는 내부 이전 중 선택 시, 입력 방식이 자동 전환된다.
  • 3. 작업 옵션: 항목을 체크하면 진행 상태를 추적할 수 있는 체크박스가 생성된다. 체크하지 않으면 ✖로 표시된다.
  • 4. 비고 입력: 메모는 직접 입력하거나 키워드 버튼을 통해 빠르게 추가할 수 있다. 입력창은 자동 확장되며, 우측 버튼으로 펼침/접기를 전환할 수 있다.
  • 5. 상태 변경: 상태 버튼 클릭 시 요청 → 진행 → 완료 순으로 변경된다.
  • 6. 현재 위치 변경: 센터 간 이전의 경우 위치 버튼 클릭 시 출발 ↔ 도착 전환이 가능하다. 출발지는 파란색, 도착지는 회색으로 표시된다.
  • 7. 작업 필터: 센터, 진행 상태에 따라 다양한 기준으로 필터링할 수 있다.
  • 8. 검색 기능: 요청자 또는 Agit 링크 기준으로 실시간 검색이 가능하다.
  • 9. 진행 사항 항목 관리: 작업이 진행됨에 따라 Unmount, 센터 이동 완료, 작업 파일, 마운트, 포설 등 세부 항목을 체크박스로 관리한다. 이를 통해 작업을 실시간으로 확인 할 수 있다.
  • 10. 수정 / 삭제: ✏️ 또는 🗑️ 아이콘을 클릭하여 작업 항목을 수정하거나 삭제할 수 있다.
  • 11. 정렬 기능: 상태, 요청일, 완료일, 벤더를 기준으로 정렬할 수 있으며, ⭡ / ⭣ 아이콘으로 정렬 방향을 표시한다.
  • 12. 통계 요약: 필터 조건에 따른 작업 수와 상태별 건수를 상단에 표시한다.
  • 13. 서버 수 통계: 센터별 서버 수를 막대 그래프로 시각화하여 작업 현황을 한눈에 파악할 수 있다.
  • 14. Excel 다운로드: 필터와 검색 조건이 반영된 데이터만 다운로드되며, 다운로드 전에 확인 모달이 표시된다.
  • 15. 비고 미리보기: 테이블 내 비고 항목 위에 마우스를 올리면 전체 메모가 툴팁으로 표시된다.
